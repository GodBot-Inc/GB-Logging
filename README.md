# GB-Logging

Simple Logging Library for the GodBot Project

## Functionalities

- Logging Modes (Info, Warning, Error)
- Logging Levels (Low, Medium, High)
- Colored Output

## Used Libraries

- [Kolor](https://github.com/ziggy42/kolor)
